# Instrucciones para Crear tu CV en PDF

## Opción 1: Usar Google Docs

1. Ve a Google Docs
2. Busca plantillas de CV profesionales
3. Completa con tu información
4. Ve a Archivo → Descargar → PDF
5. Guarda como `CV_Emilio_Guerrero.pdf`
6. Coloca el archivo en `assets/cv/`

## Opción 2: Usar Canva (Recomendado)

1. Ingresa a [Canva](https://www.canva.com)
2. Busca "CV" o "Resume" en plantillas
3. Elige una plantilla profesional
4. Personaliza con tu información
5. Descarga como PDF de alta calidad
6. Nómbralo `CV_Emilio_Guerrero.pdf`
7. Coloca el archivo en `assets/cv/`

## Opción 3: LibreOffice Writer

1. Abre LibreOffice Writer
2. Crea tu CV usando tablas y formato profesional
3. Ve a Archivo → Exportar como PDF
4. Guarda como `CV_Emilio_Guerrero.pdf`
5. Coloca el archivo en `assets/cv/`

## Opción 4: Usar LaTeX (Para usuarios avanzados)

1. Usa Overleaf o una instalación local de LaTeX
2. Utiliza una plantilla como ModernCV o AltaCV
3. Compila a PDF
4. Guarda como `CV_Emilio_Guerrero.pdf`
5. Coloca el archivo en `assets/cv/`

## Contenido Recomendado para tu CV

### Información Personal
- Nombre completo
- Título profesional
- Email
- Teléfono
- Ubicación (ciudad, país)
- LinkedIn
- GitHub

### Resumen Profesional
Un párrafo breve (3-4 líneas) destacando:
- Tu especialización (Desarrollador Backend)
- Tecnologías principales (PHP, Laravel, MySQL, PostgreSQL)
- Años de experiencia o nivel (Junior, estudiante)
- Objetivo profesional

### Experiencia Laboral
Para cada trabajo/proyecto:
- Título del puesto
- Nombre de la empresa/institución
- Fechas (mes/año - mes/año)
- Responsabilidades y logros (3-5 puntos)
- Tecnologías utilizadas

### Educación
- Título/certificación
- Institución
- Fechas
- Logros destacados (si aplica)

### Habilidades Técnicas
Organiza por categorías:
- **Backend**: PHP, Laravel, APIs REST
- **Bases de Datos**: MySQL, PostgreSQL
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap
- **DevOps**: Ubuntu Server, Git, GitHub, Docker
- **Herramientas**: VS Code, Cockpit, Nextcloud

### Proyectos Destacados
Incluye 3-4 proyectos principales:
- Nombre del proyecto
- Breve descripción (1-2 líneas)
- Tecnologías utilizadas
- Enlace al repositorio o demo (si está disponible)

### Certificaciones
Lista tus certificaciones con:
- Nombre de la certificación
- Institución emisora
- Fecha de obtención
- Horas (si aplica)

### Idiomas
- Español: Nativo
- Inglés: Nivel A2 (o el que corresponda)

## Tips para un CV Profesional

1. **Mantén el formato limpio y profesional**
   - Usa máximo 2 fuentes
   - Colores sobrios (azules, grises)
   - Espaciado consistente

2. **Longitud ideal**
   - 1-2 páginas máximo
   - Conciso y relevante

3. **Orden de secciones**
   - Información personal arriba
   - Resumen profesional
   - Experiencia (de más reciente a antigua)
   - Educación
   - Habilidades
   - Proyectos
   - Certificaciones

4. **Palabras clave**
   - Incluye tecnologías específicas
   - Usa términos de la industria
   - Optimiza para ATS (Applicant Tracking Systems)

5. **Resultados cuantificables**
   - "90% de avance en el proyecto"
   - "Gestión de base de datos con +1000 registros"
   - "Desarrollo de 5+ proyectos web"

6. **Formato PDF**
   - Asegúrate de que sea compatible
   - Tamaño de archivo menor a 2MB
   - Texto seleccionable (no escaneado)

## Verificación Final

Antes de subir tu CV, verifica:

- [ ] Nombre del archivo correcto: `CV_Emilio_Guerrero.pdf`
- [ ] Sin errores ortográficos
- [ ] Información de contacto actualizada
- [ ] Enlaces funcionando (LinkedIn, GitHub)
- [ ] Formato consistente
- [ ] Tamaño menor a 2MB
- [ ] Se ve bien en PDF reader
